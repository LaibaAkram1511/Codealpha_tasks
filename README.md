# CodeAlpha AI Internship Tasks

This repository contains the AI projects completed as part of the CodeAlpha AI Internship.

## Projects Included

### Task 1: FAQ Chatbot using NLP

## Live Demo

FAQ Chatbot:
https://faq-chatbot-laibaakram.streamlit.app

#### Description

A chatbot that answers user questions by matching them with the most relevant FAQ using Natural Language Processing techniques.

#### Technologies Used

* Python
* Streamlit
* NLTK
* Scikit-Learn
* Pandas

#### Features

* FAQ dataset handling
* Text preprocessing
* Stopword removal
* TF-IDF vectorization
* Cosine similarity matching
* Confidence score display
* Interactive chatbot interface

---

### Task 2: Vehicle Detection and Tracking

#### Description

A real-time vehicle detection and tracking system that identifies vehicles from video input and assigns tracking IDs using YOLOv8 and ByteTrack.

#### Technologies Used

* Python
* OpenCV
* YOLOv8
* ByteTrack

#### Features

* Real-time vehicle detection
* Bounding box visualization
* Vehicle classification
* Object tracking
* Tracking ID assignment
* Video processing

---

### Task 3: RAG-Based Campaign Knowledge Bot

#### Description

A Retrieval-Augmented Generation (RAG) chatbot that answers questions strictly based on provided documents using vector search and LLM-powered response generation.

#### Technologies Used

* Python
* LangChain
* ChromaDB
* Hugging Face Embeddings
* Groq API

#### Features

* PDF document ingestion
* Vector database retrieval
* Context-aware responses
* Source document references
* Similarity search
* Grounded answer generation
